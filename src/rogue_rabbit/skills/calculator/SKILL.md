# Calculator Skill

使用 Python 进行数学计算。

## 使用方式

当用户需要进行数学计算时：

1. 解析用户的数学表达式
2. 使用 Python `eval()` 执行计算
3. 返回计算结果

## 支持的运算

- **基本运算**: `+` `-` `*` `/` `//` `%`
- **幂运算**: `**`
- **比较运算**: `==` `!=` `<` `>` `<=` `>=`
- **常用函数**: `abs` `round` `min` `max` `sum`

## 示例

| 用户输入 | 计算 | 结果 |
|---------|------|------|
| "2 的 10 次方" | `2 ** 10` | 1024 |
| "100 除以 7 的余数" | `100 % 7` | 2 |
| "3.14159 四舍五入到 2 位小数" | `round(3.14159, 2)` | 3.14 |
| "1 到 100 的和" | `sum(range(1, 101))` | 5050 |

## 注意事项

- 确保表达式是合法的 Python 语法
- 对于复杂计算，可以分步执行
- 如果表达式不合法，返回错误提示
