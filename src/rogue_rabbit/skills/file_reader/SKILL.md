# File Reader Skill

读取和分析文本文件内容。

## 使用方式

当用户需要读取、分析文件时：

1. 确定文件路径
2. 读取文件内容
3. 根据用户需求进行分析

## 支持的文件类型

| 类型 | 扩展名 | 处理方式 |
|------|--------|---------|
| 文本文件 | `.txt` `.md` | 直接读取 |
| Python | `.py` | 直接读取，可分析代码结构 |
| JSON | `.json` | 解析为 Python 对象 |
| CSV | `.csv` | 按行读取，可分析数据 |
| YAML | `.yaml` `.yml` | 解析为 Python 对象 |

## 常见任务

- **快速预览**: 读取文件前 N 行
- **全文分析**: 读取完整内容并分析
- **搜索**: 在文件中搜索关键词
- **统计**: 统计行数、字数、字符数
- **格式转换**: 将文件内容转换为其他格式

## 示例

```
用户: 读取 config.json 并告诉我配置项
动作: 读取文件 -> 解析 JSON -> 列出配置项

用户: 分析 main.py 的代码结构
动作: 读取文件 -> 识别函数/类 -> 输出结构
```

## 注意事项

- 检查文件是否存在
- 注意文件编码（推荐 UTF-8）
- 对于大文件，考虑分块读取
- 不要读取二进制文件（如 .exe, .jpg）
